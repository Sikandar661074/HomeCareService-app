# Home Care Service — Customer Manager

A Flutter Android app for managing gas cylinder/home care service customers.

## Features
- Dashboard with today's renewals, due this week, and total balance
- Add / edit / delete customers
- Search by name, phone, or consumer number
- Filter by dues and renewal date range
- CSV import and export to Downloads
- Push notifications for renewals due today
- License key activation via Google Sheets

## Tech Stack
Flutter · SharedPreferences · WorkManager · flutter_local_notifications · FilePicker

## Build
```bash
flutter build apk --release
```
